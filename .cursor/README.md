# Cursor Setup for `userscripts`

This folder contains local AI guidance for working in this repository.

## What is included

- `rules/userscripts.mdc`: coding and metadata rules for `.user.js` scripts
- `rules/release-and-sharing.mdc`: release and team-sharing workflow
- `rules/security-and-permissions.mdc`: permission and security guardrails

## Why this exists

Userscripts are quick to write but easy to break for others. These rules keep:

- metadata consistent for Tampermonkey auto-updates
- permissions minimal and reviewable
- rollout predictable for teammates
